**Installing with Stow :**
```sh
sudo apt install stow
cd ~
mkdir dotfiles
cd dotfiles
git clone git@github.com:Elbtalkessel/dots.git
stow -R dots
```

**Requirements :**
- i3-gaps
- compton
- rofi
- polybar
- - at (at module)
- urxvt
- - zsh
- - - zprezto
- mpd, ncmpcpp
- neomutt
- gnome
- fonts:
- - SF, icomoon-feather, Ubuntu-C
- vim
