**Installing with Stow :**
```sh
sudo apt install stow
cd ~
mkdir dotfiles
cd dotfiles
git clone git@github.com:Elbtalkessel/dots.git
stow -R dots
```

**Requirements :**
- i3-gaps
- compton
- rofi
- polybar

**Polybar modules :**
at: requires at package
calendar: requires yad and xdotool

**I3 conf for :**
gnome, mpd

**Xresources conf for :** 
urxvt, ncmpcpp

**Misc conf for :**
mutt, zsh + zprezto

**Fonts :**
SF, icomoon-feather, Ubuntu-C
